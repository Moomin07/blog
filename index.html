GitHub Mastery Guide

Complete roadmap for using GitHub to store, manage, and showcase your web development projects

Why GitHub?

Version Control

Track changes to your code over time and revert to previous versions if needed.

Portfolio Building

Showcase your projects to potential employers or clients with a professional portfolio.

Collaboration

Work on projects with others and contribute to open source communities.

Cloud Storage

Keep your code safe in the cloud and accessible from anywhere.

Getting Started

1 Create Account

Go to github.com and sign up for a free account.

2 Install Git

Download and install Git from git-scm.com.

3 Configure Git

Set up your username and email in Git:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

4 Create Repository

On GitHub, click the "+" icon and select "New repository".

Git Commands

1 Initialize

Navigate to your project folder and run:

git init

2 Connect to GitHub

Add your GitHub repository as the remote origin:

git remote add origin https://github.com/your-username/your-repo-name.git

3 Add Files

Add your files to the staging area:

git add . # Adds all files
git add filename.html # Adds specific file

4 Commit & Push

Commit changes and push to GitHub:

git commit -m "Descriptive commit message"
git push -u origin main

Pro Tip: GitHub Pages for Free Hosting

GitHub offers free hosting through GitHub Pages. Simply enable it in your repository settings, and your website will be available at https://your-username.github.io/repository-name. Perfect for showcasing your web projects!
